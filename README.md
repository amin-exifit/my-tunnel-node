CF_TUNNEL_TOKEN → توکن تونل Cloudflare (یک رشته طولانی از پنل Zero Trust دریافت کنید)

EXIT_NODE_PSK → کلید از پیش مشترک (Pre-Shared Key) برای نود خروجی (یک رشته تصادفی انتخاب کنید)

NGROK_AUTH_TOKEN → توکن احراز هویت ngrok (از داشبورد ngrok دریافت کنید)

NGROK_DOMAIN → دامنه رزرو شده در ngrok (مثلاً myapp.ngrok.io)

TUNNEL_AUTH_KEY → کلید احراز هویت داخلی تونل (یک رشته تصادفی قوی انتخاب کنید)

توجه کنید که مقادیر EXIT_NODE_PSK و  TUNNEL_AUTH_KEY اجبارا الزامیست و باقی موارد اختیاری! (همچنین مقدار هر دو این موارد باید یکسان و برابر با مقدار کیلید کد گوگل اسکریپت باشد!)
